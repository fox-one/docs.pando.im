移除流动性

入口 1：流动池 页面 - 点击指定的池 - 流动池 “-”

入口 2：如果你已经在交易页面，在资金池部分找到“-流动性”

入口 3：如果您之前提供过流动性，您可以在“我的”页面上找到该池。 点击这个池你将看到“-”

tip

移除流动性没有手续费。
