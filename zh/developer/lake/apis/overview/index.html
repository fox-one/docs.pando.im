API概述

Pando Lake 和 4swap 为第三方开发者提供了简洁、开发者友好的 API.

目前使用的 API 端点​

名称端点
Primaryhttps://api.4swap.org/api
Alternativehttps://mtgswap-api.fox.one/api

您可以使用其中任何一个来构造您的 API 请求：

HTTP_METHOD ${API_BASE}/${API_PATH}
