读取交易对

GET /pairs#

此 API 将返回所有支持的交易对。

API 终点 URL

https://api.4swap.org/api/pairs

认证和选项

授权公开访问
限制No limitation

请求示例

curl -X GET "https://api.4swap.org/api/pairs" -H "Content-Type: application/json"
Response
{  "ts": 1627697766503,  "data": {    "pairs": [      {Pair Entity},      {Pair Entity},      ...    ],    "pair_count": 75,    "transaction_count_24h": 25134,    "volume_24h": "5189724.64904556",    "fee_24h": "13687.38904891",    "ts": 1627697766503,  }}

与 Coinmarketcap 兼容的 API#

此 API 将返回所有支持的交易对。 它符合Coinmarketcap的要求。

API 终点 URL

https://api.4swap.org/api/cmc/pairs

认证和选项

授权公开访问
限制No limitation

请求示例

curl -X GET "https://api.4swap.org/api/cmc/pairs" -H "Content-Type: application/json"
Response
