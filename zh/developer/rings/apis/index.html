API 文档

本文档描述了 Pando Rings引擎的 RESTful API 和 RPC 接口的定义和使用。

身份验证令牌#

为GET https://api.mixin.one/me签署认证令牌，范围为 "FULL "的密钥库。

请访问mixin 开发者文档以了解更多细节。

RESTful API#

• 所有市场#

返回所有可以供应和借贷的市场。

HTTP

GET /api/v1/markets/all

交易#

按时间返回所有用户交易。

HTTP

GET /api/v1/transactions?limit=10&offset=

查询参数

key类型des
limitstring页面数据上限，默认500个
offsetstring分页开始时间，RFC3339Nano 格式，例如：2020-12T12:12:12.99999999999 Z

}
