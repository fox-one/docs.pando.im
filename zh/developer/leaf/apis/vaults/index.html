读取所有金库#

GET /vats#

此 API 将返回所有已创建的金库。

API 终点 URL

https://leaf-api.pando.im/api/vats

认证和选项

授权公开访问
限制No limitation

参数

cursorthe cursor to start from
limitthe limitation of items in response

curl -X GET "https://leaf-api.pando.im/api/vats" -H "Content-Type: application/json"

读取单个金库#

GET /vat/:id#

此 API 将根据 :id 返回单个金库。

API 终点 URL

https://leaf-api.pando.im/api/vats/:id

认证和选项

授权公开访问
限制No limitation

参数

id
*标注的为必填项目
the vault id

请求示例

curl -X GET "https://leaf-api.pando.im/api/vats/52010fd0-6a9d-393c-abcb-ca997d950bf5" -H "Content-Type: application/json"
Response
{  "ts": 1633140496485,  "data": {    "id": "52010fd0-6a9d-393c-abcb-ca997d950bf5",    "created_at": "2021-04-27T10:51:16Z",    "collateral_id": "d0ec4cc7-edf6-5359-bf23-41fc9d26444e",    "ink": "2",    "art": "30530.6925452775230346",    "identity_id": "1"  }}

读取金库的所有事件#

GET /vat/:id/events#

此 API 将根据 :id 返回一个金库的所有事件。

API 终点 URL

https://leaf-api.pando.im/api/vat/:id/events

认证和选项

授权公开访问
限制No limitation

参数

id
*标注的为必填项目
the vault id

请求示例

curl -X GET "https://leaf-api.pando.im/api/vaults/52010fd0-6a9d-393c-abcb-ca997d950bf5/events" -H "Content-Type: application/json"
Response

读取我的金库#

GET /me/vats#

此 API 将返回当前用户的所有金库。

API 终点 URL

https://leaf-api.pando.im/api/me/vats

认证和选项

授权
Authorized
限制No limitation

参数

cursorthe cursor to start from
limitthe limitation of items in response

请求示例

curl -X GET "https://leaf-api.pando.im/api/me/vats" -H "Content-Type: application/json" -H "Authorization: Bearer $TOKEN"
Response
{  "ts": 1633140121000,  "data": {    "vaults": [      {        "id": "52010fd0-6a9d-393c-abcb-ca997d950bf5",        "created_at": "2021-04-27T10:51:16Z",        "collateral_id": "d0ec4cc7-edf6-5359-bf23-41fc9d26444e",        "ink": "2",        "art": "30530.6925452775230346",        "identity_id": "1"      },      // ...    ],    "pagination": {      "next_cursor": "100",      "has_next": true    }  }}
