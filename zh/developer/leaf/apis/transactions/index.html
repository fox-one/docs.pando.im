Read Transactions

Read All Transactions#

GET /transactions#

This API will respond all transactions.

Endpoint URL

https://leaf-api.pando.im/api/transactions

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Example request

curl -X GET "https://leaf-api.pando.im/api/transactions" -H "Content-Type: application/json"

Read Single Transaction#

GET /transactions/:follow_id#

This API will respond one transaction with :follow_id. The follow_id is an uuid that you specified when you created the action.

Endpoint URL

https://leaf-api.pando.im/api/transactions/:follow_id

Authentication and options

Authorization
Authorized
LimitationNo limitation

Parameters

follow_id
*required
the follow id

Example request

curl -X GET "https://leaf-api.pando.im/api/transactions/c8c92c8f-65b3-49b7-bfae-d5ae43265129" -H "Content-Type: application/json" -H "Bearer: $TOKEN"
Response
