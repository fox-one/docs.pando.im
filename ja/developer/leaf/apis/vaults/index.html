Read All Vaults​

GET /vats​

This API will respond all vaults.

Endpoint URL

https://leaf-api.pando.im/api/vats

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Parameters

limitthe limitation of items in response

Read Single Vault​

GET /vat/:id​

This API will respond one vault with :id

Endpoint URL

https://leaf-api.pando.im/api/vats/:id

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Parameters

id
*required
the vault id

Example request

curl -X GET "https://leaf-api.pando.im/api/vats/52010fd0-6a9d-393c-abcb-ca997d950bf5" -H "Content-Type: application/json"
Response
{
  "ts": 1633140496485,
  "data": {
    "id": "52010fd0-6a9d-393c-abcb-ca997d950bf5",
    "created_at": "2021-04-27T10:51:16Z",
    "collateral_id": "d0ec4cc7-edf6-5359-bf23-41fc9d26444e",
    "ink": "2",
    "art": "30530.6925452775230346",
    "identity_id": "1"
  }
}

Read All Vault Events​

GET /vat/:id/events​

This API will respond all events of one vault with :id

Endpoint URL

https://leaf-api.pando.im/api/vat/:id/events

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

the vault id

Read My Vaults​

GET /me/vats​

This API will respond all vaults belonging to current user.

Endpoint URL

https://leaf-api.pando.im/api/me/vats

Authentication and options

Authorization
Authorized
LimitationNo limitation

Parameters

cursorthe cursor to start from
limitthe limitation of items in response

Example request

curl -X GET "https://leaf-api.pando.im/api/me/vats" -H "Content-Type: application/json" -H "Authorization: Bearer $TOKEN"
Response
{
  "ts": 1633140121000,
  "data": {
    "vaults": [
      {
