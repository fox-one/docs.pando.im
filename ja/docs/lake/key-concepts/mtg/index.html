4swap MTG

4swap is built with Mixin Trusted Group. Right now, there are 5 genesis nodes that verify the transactions with Mixin Network and performs administrative actions, they are:

  • Mixin Team: The core develop team of Mixin Network
  • Fox.ONE: The main development team of 4swap
  • Poolin: The industry-leading mining service provider
  • Big.ONE: A global trading platform that provides users with secure and convenient trusteeship services
  • BOX group: Li Xiaolai’s investment community
