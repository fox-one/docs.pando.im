Read Info

Read MTG Info#

GET /info#

이 API는 Pando Leaf의 MTG 정보에 응답합니다.

info

MTG 정보가 자주 변경되지 않으므로 나중에 사용하기 위해 응답을 저장할 수 있습니다.

엔드포인트 URL

https://leaf-api.pando.im/api/info

인증 및 옵션

권한 부여공용 액세스
제한 사항No limitation

예시 요청

curl -X GET "https://leaf-api.pando.im/api/info" -H "Content-Type: application/json"
Response

Read Server Time#

GET /time#

이 API는 Pando Leaf의 서버 시간에 응답합니다.

엔드포인트 URL

https://leaf-api.pando.im/api/time

인증 및 옵션

권한 부여공용 액세스
제한 사항No limitation

예시 요청

curl -X GET "https://leaf-api.pando.im/api/time" -H "Content-Type: application/json"
Response
{  "ts": 1633139971226,  "data": {    "iso": "2021-10-02T01:59:31Z",    "epoch": 1633139971  }}
