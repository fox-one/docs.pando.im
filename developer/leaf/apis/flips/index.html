Read Auctions

Read All Auctions​

GET /flips​

This API will respond all auctions.

Endpoint URL

https://leaf-api.pando.im/api/flips

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Parameters

cursorthe cursor to start from
limitthe limitation of items in response

Example request

curl -X GET "https://leaf-api.pando.im/api/flips" -H "Content-Type: application/json"
Response
{
  "flips": [
    {
      // ActionKick: the auction begins, any bid are accepted
      // ActionBid: the auction bidding
      // ActionDeal: the auction is over
      "action": 0,
      "art": "string",
      // the current amount of bid
      "bid": "string",
      "collateral_id": "string",
      "created_at": "2021-10-02",
      // auction end time
      "end": "2021-10-02",
      "guy": "string",
      "id": "string",
      // the amount of auctioned collateral
      "lot": "100",
      // the max amount of bid accepted
      "tab": "2",
      // bid end time
      "tic": "2021-10-02",
      "vault_id": "string"
    }
  ],
  "pagination": {
    "has_next": true,
    "next_cursor": "string"
  }
}

Read Single Auctions​

GET /flips/:id​

This API will respond one auction by :id.

Endpoint URL

https://leaf-api.pando.im/api/flips/:id

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Example request

curl -X GET "https://leaf-api.pando.im/api/flips/4ce8a961-4b90-34e8-9780-6ce46a8205a6" -H "Content-Type: application/json"
Response

  "data": {
    "id": "4ce8a961-4b90-34e8-9780-6ce46a8205a6",

Read Auction Events​

GET /flips/:id/events​

This API will respond all events of one auction by :id.

Endpoint URL

https://leaf-api.pando.im/api/flips/:id/events

Authentication and options

AuthorizationPublic Access
LimitationNo limitation

Example request

curl -X GET "https://leaf-api.pando.im/api/flips/4ce8a961-4b90-34e8-9780-6ce46a8205a6/events" -H "Content-Type: application/json"
